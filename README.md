 Flappy Bird Game (Python)

This is a small Flappy Bird type game that I made using Python.
I created this while practicing basic programming and trying to understand how simple games work.

It is not perfect, but it was a fun project and helped me learn things like loops, conditions, and basic game logic.



 What this game has

* Bird that moves with gravity
* Jump control using spacebar
* Pipes coming from right side
* Score system
* Game over when collision happens



 How to run it

1. Make sure Python is installed on your system
2. Download or clone this project
3. Open the folder in VS Code or any editor
4. Run this command:


python flappy.py



Controls

Press SPACEBAR to make the bird jump
